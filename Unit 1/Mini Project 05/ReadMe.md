# Mini Project 5: Restaurant Ordering Interface

## Project Overview

The Restaurant Ordering Interface is a simple HTML-based web application that demonstrates how a restaurant can display its menu and allow customers to place food orders online.

This mini project is designed to help students understand the practical usage of various HTML elements such as tables, forms, input controls, lists, hyperlinks, and page navigation.

## Project Objective

The main objective of this project is to create a user-friendly interface where customers can:

* View available food items and prices.
* Enter their personal details.
* Select food items from the menu.
* Choose order type and payment method.
* Submit their order successfully.

## Project Structure

The project consists of two web pages:

### 1. Restaurant Ordering Page

This is the main page where customers can:

* View the restaurant menu.
* Enter customer information.
* Select food items using checkboxes.
* Choose order type using radio buttons.
* Select a payment method from a dropdown list.
* Provide special instructions.
* Place or reset the order.

### 2. Order Confirmation Page

After clicking the "Place Order" button, the user is redirected to a confirmation page that:

* Displays a successful order message.
* Confirms that the order has been received.
* Shows the order status as confirmed.
* Provides an option to place another order.

## HTML Concepts Used

This project demonstrates the use of:

* HTML document structure
* Headings and paragraphs
* Tables
* Forms
* Text input fields
* Text areas
* Checkboxes
* Radio buttons
* Dropdown lists
* Submit and reset buttons
* Hyperlinks
* Lists

## Learning Outcomes

After completing this project, students will be able to:

* Design a basic restaurant ordering interface.
* Create and use HTML forms effectively.
* Collect user information through form controls.
* Navigate between multiple web pages.
* Organize content using tables and lists.
* Develop a simple real-world web application using HTML.

## Conclusion

This mini project provides hands-on experience in building a simple restaurant ordering website using HTML. It helps students understand how different HTML elements work together to create an interactive and structured web page.
